Welcome to Employee-Department REST-API !!


This API helps to search the list of employees and their details based on the name of their department.

The 4 departments whose employee details can be called through this API is as follows:
  • FINANCE
  • AUDIT
  • PRODUCTION
  • MARKETING

URL to call the api :
http://127.0.0.1:5000/department/q=department_name
Examples :
  • http://127.0.0.1:5000/department/q=FINANCE
  • http://127.0.0.1:5000/department/q=AUDIT
  • http://127.0.0.1:5000/department/q=PRODUCTION
  • http://127.0.0.1:5000/department/q=MARKETING


OUTPUT/Response: The response of the queries will be in form of JSON.
