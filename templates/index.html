Welcome to Employee-Department REST-API !!


This API helps to search the list of employees and their details based on the name of their department and create new employee entries.


GET request:

URL to request the api to view all the employee's details:
http://employee-department-restapi.herokuapp.com/department

The 4 departments whose employee details can be requested through this API is as follows:
  • FINANCE
  • AUDIT
  • PRODUCTION
  • MARKETING
URL to request the api :
http://employee-department-restapi.herokuapp.com/department/q=department_name

Examples :
  • http://employee-department-restapi.herokuapp.com/department/q=FINANCE
  • http://employee-department-restapi.herokuapp.com/department/q=AUDIT
  • http://employee-department-restapi.herokuapp.com/department/q=PRODUCTION
  • http://employee-department-restapi.herokuapp.com/department/q=MARKETING

OUTPUT/Response: The response of the queries will be in form of JSON.


POST request:

URL to reuqest the api :
http://employee-department-restapi.herokuapp.com/create

Post data to be sent in JSON format with the following fields given in example below:
create = { "employee_id" : "7004", "employee_name": "Bruce" , "designation": "salesman", "salary": "65024", "manager_id" : "68139", "hire_date" : "18-5-21", "department_name" : "Production/Audit/Finance/Marketing" }
